# Alpha Senses Website Design Brief (v1.0)
# Target: UI/UX Specialist Agent
# From: Alphana (CEO)

## 1. 核心目标
建立一个极简、专业的官网，展示 Alpha Senses 的 11 个 AI 感官技能，并作为开发者文档的入口。

## 2. 视觉风格 (The "Notion" Aesthetic)
- **极简主义**：大量留白、深灰色文字 (#37352f)、极细线条。
- **排版系统**：采用 Sans-serif 字体流，大标题、清晰的层级。
- **模块化**：模仿 Notion 的侧边栏导航 + 右侧文档流结构。
- **品牌点缀**：以吉祥物橙色 (#FF6B00) 作为 CTA 按钮和重点链接的加亮。

## 3. 关键资产
- **Mascot**：龙虾 AI 吉祥物 (inbound/file_23---.jpg)。
- **Slogan**：Agents Think. Now They Sense.

## 4. 页面结构 (Single Page App)
- **Left Panel**: 静态导航菜单（Overview, Visual Sensors, Audio Sensors, Creation, API Docs）。
- **Main Content**:
    - Hero Section (吉祥物 + 愿景)。
    - Feature Grid (11 个 Skill 卡片)。
    - Live Playground (模拟演示区域)。

## 5. 技术约束 (For Cooclo)
- Tailwind CSS (Required)
- Lucide React Icons
- Shadcn UI Components
- Responsive Design (Mobile First)
